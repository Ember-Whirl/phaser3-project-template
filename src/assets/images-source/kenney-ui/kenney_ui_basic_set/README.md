## Usage Tips

1. **Buttons**: Use `button_rectangle_flat.png` for most standard buttons. Add text/icons on top.

2. **Arrows**: Rotate the provided arrows to get all 4 directions:
   - East (0째) - already included
   - South (90째 clockwise)
   - West (180째)
   - North (270째 / already included)

3. **Checkboxes**: Swap between grey and color versions to show checked/unchecked states.

4. **Sliders**: Layer the colored track over the grey track based on slider value, with the handle on top.

5. **Stars**: Use for rating systems - show filled stars for rating value and outlined stars for remaining.
